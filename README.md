# Projeto Herança em Java

Este projeto foi desenvolvido com o objetivo de praticar os conceitos de Programação Orientada a Objetos (POO), com foco em herança em Java.

Foram utilizadas classes como Pessoa, Aluno, Professor e Funcionário para demonstrar o uso de herança e reutilização de código.

## Conceitos aplicados

- Herança
- Encapsulamento
- Uso de construtores com super()

  ## Estrutura

- Pessoa (classe MÃE)
- Aluno (herda de Pessoa)
- Professor (herda de Pessoa)
- Funcionário (herda de Pessoa)
- Principal (classe de execução)

  ## Funcionalidades

O sistema simula diferentes tipos de pessoas em um contexto acadêmico, demonstrando como atributos e métodos são herdados e modificados nas subclasses.

## Observação
Este é um projeto de estudo e faz parte da minha evolução como desenvolvedor backend. 
Pretendo evoluir este projeto futuramente aplicando esses conceitos em APIs com Spring Boot.
